Getting Started

Welcome to KIP! Here are some basic gestures to help you navigate the application:

Actions Gestures Keyboard Shortcuts
Cycle through dashboards Swipe Up/Down Shift + Ctrl + Up/Down Arrow
Open Actions menu Swipe left Shift + Ctrl + Right Arrow
Open Notification menu Swipe right Shift + Ctrl + Left Arrow

Load Demonstration Dashboards

Try the sample Dashboard with simulated data from demo.signalk.org (internet required to receive data).

Create Your Own Layout

First, configure Signal K server authentication (a pre-existing Signal K server user/password is required). Swipe left to open the Actions menu, hit Settings, toggle Login to server and enter you pre-existing Signal K credentials.

Dashboards are "Locked" by default to prevent accidental modifications. To edit dashboard a, swipe left and hit the bottom unlock menu button. You can later add and manage dashboards using the Dashboards from the Actions menu.

Build Your Layout

First, resize this Tutorial widget using the now visible dashed borders so it does not take up all the screen. Then, to add a widget: in an empty dashboard area, press and hold. This will open the widget selection panel. Here you can choose from a variety of widgets.

To manage widgets, you can:

  • Touch and drag to move widgets
  • Drag corners and sides to resize widgets
  • Double Tap on a widget to configure it
  • Press and Hold on a widget to delete or duplicate widgets

Exit Dashboard Edit Mode

Use the Save or Cancel buttons in the lower right corner to either persist your changes or revert changes.

Add More Dashboards

Swipe left and hit the Dashboards button. From this page you can add, rename, position and duplicate dashboards.
